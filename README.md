# go-rank-estimator

## How to run

Start walking, then start walking more quickly by moving your legs fast until you end up kinda jumping and propelling yourself forward with only one leg on the ground at a time.

## How to get this program to run:

1. Connect to db via ssh
2. Export environment variables (DB_USER, DB_PASSWORD, DB_PORT)
3. The command to run is just `tsx index.ts` while in the `src` directory. If your machine does not recognize the `tsx` command, you can install it globally with `npm i -g tsx`, but the package is a dev dependency, so it should already be available.
